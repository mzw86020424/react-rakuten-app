# 環境構築

1. このリポジトリをクローンする
```bash
git clone https://github.com/mzw86020424/react-rakuten-app.git
```
2. クローンしたリポジトリに移動する
```bash
cd app
```
3. Dockerイメージをビルドする
```bash
docker-compose build
```
4. Dockerコンテナを起動する
```bash
docker-compose up
```
5. ブラウザでアプリケーションを開く
http://localhost:3000にアクセスして、アプリケーションを閲覧することができます。
